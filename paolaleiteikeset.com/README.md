# Paola Leite Eikeset Advocacia

Projeto React/Vite pronto para publicar na Vercel.

## Como publicar no GitHub pelo navegador

1. Crie um repositório no GitHub.
2. Faça upload de todos os ficheiros desta pasta.
3. Na Vercel, clique em **Import Git Repository**.
4. Framework Preset: **Vite**.
5. Build Command: `npm run build`.
6. Output Directory: `dist`.
7. Deploy.

## Como testar no computador

```bash
npm install
npm run dev
```
